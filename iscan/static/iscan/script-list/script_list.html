Scripts

This page is dedicated to the running of SPADE scripts. These are Python scripts which are written to directly interface with PolyglotDB and the ISCAN server without using the front-end user interface. For security reasons, it is not safe to allow people to upload their own scripts, however, if you write one, you can contact the server administrator. They will then be able to put the script on the server directly.
